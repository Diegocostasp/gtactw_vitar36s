# GTA Chinatown Wars PortMaster

This is an experimental PortMaster loader for the Android ARMv7 version of GTA Chinatown Wars.

Required files are not included.

Place your legally obtained files in:

gtactw/data/

Required:

- libCTW.so from APK lib/armeabi-v7a/
- assets folder from APK
- extracted OBB contents
